# Life Tracker Mobile

Android-приложение на Kotlin с использованием Jetpack Compose для отслеживания задач и прогресса героя.

## Технологии

- **Jetpack Compose** - для всего UI
- **MVVM** - архитектура приложения
- **Room** - локальная база данных
- **Retrofit** - для работы с API
- **Coroutines** - для асинхронных операций
- **Navigation Compose** - для навигации между экранами

## Функциональность

- Отображение карточки героя с уровнем и прогрессом опыта
- Список активных задач
- Добавление новых задач
- Выполнение задач с начислением опыта
- Оффлайн режим: задачи сначала сохраняются локально, затем синхронизируются с сервером
- Автоматическая синхронизация несинхронизированных задач при запуске

## Настройка

1. Убедитесь, что сервер запущен на `http://10.0.2.2:5000/api/Task`
2. Для эмулятора Android используется `10.0.2.2` вместо `localhost`
3. Приложение настроено для работы с HTTP трафиком (network security config)

## Структура проекта

```
app/src/main/java/com/lifetracker/mobile/
├── api/              # Retrofit API клиент
├── data/             # Модели данных и Room
├── navigation/        # Навигация Compose
├── repository/        # Репозитории для работы с данными
├── ui/               # Compose UI компоненты
│   ├── components/   # Переиспользуемые компоненты
│   ├── screens/      # Экраны приложения
│   └── theme/        # Тема приложения
└── viewmodel/        # ViewModel для MVVM
```

## Запуск

1. Откройте проект в Android Studio
2. Синхронизируйте Gradle файлы
3. Запустите приложение на эмуляторе или устройстве
